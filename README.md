### Machine Learning Project for Early Diabetes Detection

**Diabetes Prediction** is a machine learning project that predicts whether a person is likely to have **diabetes** based on several medical and lifestyle features.  
The project uses **data preprocessing, exploratory data analysis (EDA), and supervised machine learning models** to perform binary classification.

This project is implemented using **Python** and presented in a **Jupyter Notebook**.

- Build predictive models to assist early diagnosis
